# Projektarbeit 27.11. bis Ende Semester (15.1.2018) <br /> Alleine oder zu zweit ## Ziel * Mit Javascript ein kleines Spiel programmieren ## Ablauf Jede Woche werden 2-3 Code-Komponenten vorgestellt, die für die Programmierung verwendet werden können. ## Kriterien Der Code muss selber geschrieben werden. Es ist besser, ein kleines eigenes Projekt abzugeben als ein grosses kopiertes. Funktional: * Das Spiel funktioniert und ist in mindestens zwei der folgenden Browsern spielbar (Firefox, Chrome, Internet Explorer) [5 Punkte] * Das Ziel des Spiels ist ersichtlich und verständlich [3 Punkte] * Das Spiel hat eine Anzeige der Punkte, die der Spieler erreicht hat [2 Punkte] * Die Steuerung fühlt sich natürlich an [2 Punkte] * Grafiken und Layout sind sorgfältig gemacht [2 Punkte] * Das Spiel macht Spass [2 Punkte] Nicht-funktional: * Variablennamen beschreiben deren Inhalt, Funktionsnamen deren Tätigkeit [2 Punkte] * Der Code ist entsprechend dem [Style Guide](https://google.github.io/styleguide/jsguide.html#formatting) formatiert. Wichtig: Inhalte von Funktionen, Bedingungen und Schleifen einrücken. [2 Punkte] Total: 20 Punkte ## Ideen * Snake * Sterne fangen * Flappy Bird * Tic Tac Toe * Pong ## Abgabe Einen ZIP-Ordner pro Gruppe. Im Ordner müssen alle Dateien, Bilder und Sounds enthalten sein. Ausserdem muss der Ordnername die Namen der Gruppenmitgliedern enthalten. Abgabe bis 15.1.2018 per E-Mail an lukas.diener@bbz-biel.ch. Bitte Dateien nicht anhängen sondern auf einen Filehoster hochladen und den Link senden.
