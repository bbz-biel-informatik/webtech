# Projektarbeit <br /> Alleine oder zu zweit ## Ziel * Mit Javascript ein kleines Spiel programmieren ## Ablauf Jede Woche werden 2-3 Code-Komponenten vorgestellt, die für die Programmierung verwendet werden können. Im letzten Kurs werden die Games der Klasse vorgestellt. ## Kriterien Der Code muss selber geschrieben werden. Es ist besser, ein kleines eigenes Projekt abzugeben als ein grosses kopiertes. **Vortrag**: * Dauer: 5-7 Minuten * Präsentation Gameidee * Wie werden Punkte erzielt? * Steuerung * Code * Verwendete Bibliotheken (wozu werden diese gebraucht?) * Codeausschnitt * Schwiergikeiten & Verbesserungsvorschläge * Fragen zum Code **Kriterien Projekt**: * Das Spiel funktioniert und ist in den Browsern Chrome und Firefox spielbar * Das Ziel des Spiels ist ersichtlich und verständlich * Das Spiel hat eine Anzeige der Punkte, die der Spieler erreicht hat * Die Steuerung fühlt sich natürlich an * Grafiken und Layout sind sorgfältig gemacht * Das Spiel macht Spass * Variablennamen beschreiben deren Inhalt, Funktionsnamen deren Tätigkeit ## Ideen * Snake * Sterne fangen * Flappy Bird * Pong ## Abgabe Einen ZIP-Ordner pro Gruppe. Im Ordner müssen alle Dateien, Bilder und Sounds enthalten sein. Ausserdem muss der Ordnername die Namen der Gruppenmitgliedern enthalten.
