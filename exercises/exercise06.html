# Events ## Aufgabe 0 Schreibe Code, der beim Klick auf die Überschrift deiner Website diese Überschrift umfärbt. ## Aufgabe 1 Schreibe Code, der beim Klicken auf ein Bild dieses Bild vergrössert. Ein erneuter Klick soll das Bild wieder verkleinern. ## Aufgabe 2 Schreibe eine Website, die ein Formular beinhaltet. Sobald der Benutzer mit der Maus über den "Absenden"-Knopf fährt, soll dieser an eine zufällige Position auf der Website springen. ## Aufgabe 3 Schreibe Javascript-Code, der *jedem* Element beim Drüberfahren mit der Maus einen roten Rahmen macht. Du kannst den CSS-Selektor `*` dazu verwenden. Bonus: Wenn du die Maus über ein Element bewegst, wird der `mouseover`-Event automatisch auch für alle Eltern-Elemente ausgeführt. Wenn du diese nicht färben möchtest, kannst du verhindern, dass sich der Event weiterverbreitet. Dazu musst du deine Handler-Funktion etwas umschreiben und bei der Funktionsdefinition einen neuen Parameter `e` angeben: Alt: ```js function färben() { //dein Code hier } ``` Neu: ```js function färben(e) { //dein Code hier e.stopPropagation(); } ```
