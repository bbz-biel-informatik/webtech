# Events ## Aufgabe 1 Schreibe Code, der beim Klicken auf ein Bild dieses Bild vergrössert. Ein erneuter Klick soll das Bild wieder verkleinern. ## Aufgabe 2 Schreibe eine Website, die ein Formular beinhaltet. Sobald der Benutzer mit der Maus über den "Absenden"-Knopf fährt, soll dieser an eine zufällige Position auf der Website springen. ## Aufgabe 3 Schreibe Javascript-Code, der *jedem* Element beim Drüberfahren mit der Maus einen roten Rahmen macht. Du kannst den CSS-Selektor `*` dazu verwenden. Bonus: Wenn du die Maus über ein Element bewegst, wird der `mouseover`-Event automatisch auch für alle Eltern-Elemente ausgeführt. Wenn du diese nicht färben möchtest, kannst du verhindern, dass sich der Event weiterbewegt. Dazu musst du deine Handler-Funktion etwas umschreiben und bei der Funktionsdefinition einen neuen Parameter `e` angeben: Alt: ```js function färben() { //dein Code hier } ``` Neu: ```js function färben(e) { //dein Code hier e.stopPropagation(); } ``` ## Lösungen ### Aufgabe 1 ```js var image = document.querySelector('img') image.addEventListener('click', scale); var large = false; function scale() { if(large) { image.style.width = ''; large = false; } else { image.style.width = '80%'; large = true; } } ``` ### Aufgabe 2 ```js <&nbsp;form> <&nbsp;input type="text"/> Benutzername <&nbsp;input type="text"/> Email <&nbsp;input type="submit"/> <&nbsp;/form> var submit = document.querySelector("input[type=submit]"); submit.addEventListener('mouseover', move); function move() { submit.style.position = 'absolute'; submit.style.top = Math.random() * 200 + 'px'; submit.style.left = Math.random() * 200 + 'px'; console.log(submit.style.left + ' ' + submit.style.top) } ``` ### Aufgabe 3 ```js var all = document.querySelectorAll("*"); for(var i = 0; i < all.length; i++) { all[i].addEventListener('mouseover', changeColor); all[i].addEventListener('mouseout', changeColorBack); } function changeColor(e) { this.style.border = '1px solid red'; e.stopPropagation(); } function changeColorBack(e) { this.style.border = 'none'; e.stopPropagation(); } ```
