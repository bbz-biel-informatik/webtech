# Events ## Aufgabe 0 Schreibe Code, der sofort beim Laden der Seite die Hintergrundfarbe auf rot ändert. ## Aufgabe 1 Schreibe Code, der beim Klick auf die Überschrift deiner Website diese Überschrift umfärbt. ## Aufgabe 2 Schreibe Code, der beim Klicken auf ein Bild dieses Bild vergrössert. ### Aufgabe 2.1 Erweitere deinen Code so, dass ein erneuter Klick das Bild wieder verkleinert. ## Aufgabe 3 Schreibe eine Website, die ein Formular beinhaltet. * Sobald der Benutzer in eine Eingabefeld klickt, soll dieses einen roten Rahmen erhalten * Sobald der Benutzer mit der Maus über den "Absenden"-Knopf fährt, soll dieser an eine zufällige Position auf der Website springen. ## Aufgabe 4 Schreibe Javascript-Code, der *jedem* Element beim Draufklicken einen roten Rahmen macht. Du kannst den CSS-Selektor `*` dazu verwenden.
