## Akkordeon Baue auf deiner Website ein Akkordeon-Menu ein. Style das Menu dann mit CSS, damit es einigermassen brauchbar aussieht. ## Tastatur Versuche die Website so zu erweitern, dass man sie mit der Tastatur navigieren kann. Das Drücken der Taste 'e' soll zum Beispiel ans Ende der Seite springen, und das Drücken einer Zahl soll zur entsprechenden Überschrift springen (z.B. '3' springt zur 3. Überschrift). ## Tabs Baue dein Akkordeon so um, dass es aus Tabs anstatt aus ausklappenden Elementen besteht.
