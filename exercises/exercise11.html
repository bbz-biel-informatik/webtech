## Tastatur Versuche die Website so zu erweitern, dass man sie mit der Tastatur navigieren kann. Das Drücken der Taste 'e' soll zum Beispiel ans Ende der Seite springen, die Taste 'b' zum Beginn und das Drücken einer Zahl soll zur entsprechenden Überschrift springen (z.B. '3' springt zur 3. Überschrift). Füge auch für Aktionen, die bishermit einem Button gestartet wurden, noch Tastaturbefehle hinzu. ## Benachrichtigungen Baue auf deiner Website einen Timer ein, der dich nach 10 Sekunden Inaktivität fragt, ob du noch wach seist. Du kannst den folgenden Code als Grundgerüst verwenden: ```js // nach 10 Sekunden Nachricht anzeigen var timer = window.setTimeout(nachricht, 10000); // beim Scrollen und Maus-bewegen Timer neu starten document.addEventListener('scroll', resetTimer); document.addEventListener('mousemove', resetTimer); function resetTimer() { // aktuellen Timer löschen und neu starten window.clearTimeout(timer); timer = window.setTimeout(nachricht, 10000); } function nachricht() { // ... dein Code } ```
