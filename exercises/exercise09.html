# Uhr Baue auf deiner Website eine Uhr ein. Alle 10 Sekunden soll dein Code die aktuelle Uhrzeit abrufen und auf der Website anzeigen. Du kannst den folgenden Code verwenden, um die Uhrzeit herauszufinden: ```js var zeit = new Date(); var stunden = zeit.getHours(); var minuten = zeit.getMinutes(); ``` In den Variablen `stunden` und `minuten` ist dann die aktuelle Stunde bzw. Minute gespeichert. Bonus: zeige auch noch die Sekunden an! # Bild Programmiere deine Website so, dass das erste Bild deiner Website automatisch nach rechts wandert. Halte die Animation an, sobald das Bild am rechten Rand angekommen ist. Tipp: Schau in den Folien das erste Beispiel an, und passe es so an, dass nicht die Breite, sondern z.B. der linke Margin geändert wird. # Scroll-Animation Baue die Scrollfunktion deiner Website so um, dass das Scrollen animiert ist. Gehe am besten so vor: * Speichere die Position des Ziel-Elementes: `var ende = document.querySelector('#p1').offsetTop;` * Schreibe eine Animation wie in den Folien * Stoppe die Animation, wenn `ende` erreicht ist # Schneefall Simuliere Schneefall auf deiner Website. Generiere dazu an zufälligen Positionen am oberen Rand der Website kleine divs, die dann herunterfallen. Verwende den folgenden Code, wenn du nicht weiterkommst: https://jsfiddle.net/e8vkkb9j. Er generiert den Schnee, du musst ihn nur noch fallen lassen. Verwende weiteres CSS und ändere die Intervalle, damit der Schnee realistischer aussieht.
