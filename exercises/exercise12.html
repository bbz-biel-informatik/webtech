# Karten &amp; Standort ## Karten Baue auf deiner Website eine Google-Karte ein. Du kannst den Code vom Beispiel, das in den Folien verlinkt ist, kopieren. Ändere die folgenden Eigenschaften: * Stelle den Kartenausschnitt so ein, dass dein Zuhause zu sehen ist * Zoome auf eine geeignete Zoomstufe * Bearbeite deinen CSS-Code so, dass die Karte einigermassen schön in dein Layout eingefügt ist ## Marker * Füge Marker für die wichtigsten Orte hinzu, z.B. den nächsten Kiosk oder den Bancomaten. Beschrifte die Marker sinnvoll ## Standortabfrage * Füge einen Marker beim Standort des Websitebesuchers ein.
