[← Übungen 3](exercise03.html) ## Treppe Im folgenden Zeichungsfeld kann über die Konsole mit den Befehlen `right()`, `left()`, `up()` und `down()` gezeichnet werden. Der Code ```js right() up() left() down() ``` zeichnet zum Beispiel ein Quadrat. Der Befehl `reset()` löscht das gezeichnete und setzt den Stift auf die Ausgangsposition. ### Aufgabe * Schreibe ein Programm, das drei Stufen Zeichnet. * Schreibe ein möglichst kurzes Programm, das 15 Stufen zeichnet (4 Zeilen Code reichen aus)
