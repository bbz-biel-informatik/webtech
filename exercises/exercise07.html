# Elemente generieren und löschen ## Websitevorschau Versuche eine Website zu bauen, die mehrere Links zu externen Seiten und einen einzigen iframe beinhaltet. Sobald ein Besucher auf einen Link klickt, soll dieser nicht geöffnet, sondern im iframe angezeigt werden. Um eine Website in einem iframe anzuzeigen, kannst du dessen `src`-Attribut auf die Website-URL setzen. Achtung: einige Websites lassen es nicht zu, dass sie in einem iframe geladen werden (z.B. Google). In diesem Fall siehst du in der Javascript-Konsole eine entsprechende Meldung. ## Navigation 1997-Style Baue eine Website, die oben eine Navigation mit Untermenupunkten anzeigt. Das Menu soll die folgende Struktur haben: Home Seiten - Über mich - Gästebuch Sitemap Sobald Benutzer mit der Maus über den Eintrag "Seiten" fahren, soll ein Untermenu ausklappen, das dessen Unterseiten anzeigt. Am einfachsten geht das, wenn du im HTML-Code alle Menupunkte definierst und sie per JS anzeigst und versteckst. ## Diashow Baue eine Website, die eine Diashow beinhaltet. Die Diashow soll zum nächsten Bild wechseln, sobald die Besucher auf das Bild klicken. Am einfachsten geht das, indem du ein Array mit den Internet-Adressen der Bilder speicherst und beim Klick jeweils ein neues Bild-Element generierst, das die entsprechende Adresse als `src`-Attribut hat. Dann positionierst du das Bild gleich wie das vorherige.
