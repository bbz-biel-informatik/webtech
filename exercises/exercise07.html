# Events ```js var x = document.querySelector('img') for(var i = 0; i < x.length; i = i + 1) { x[i].addEventListener("click", randHinzufügen) } function randHinzufügen() { this.style.border = '5px solid blue' } ``` ## Bilderrahmen Baue auf deiner Website ein, dass ein Klick auf ein beliebiges Bild diesem einen blauen Rahmen gibt ## Bilderrahmen 2 Erweitere die Bilderrahmenaufgabe so, dass ein weiterer Klick auf ein beliebiges Bild diesem den Rahmen wieder wegnimmt ## Bilderposition Baue ein, dass sobald ein Besucher mit der Maus über ein Bild fährt, sich dieses wegverschiebt. Bonus: verschiebe es jedes Mal, wenn man mit der Maus drüberfährt an eine neue zufällige Position. Verwende dazu statt dem `click`-Ereignis das `mouseover`-Ereignis. ## Fokus Baue auf deiner Website ein kleines Formular ein (siehe https://jsfiddle.net/s1zuev5L/). Sobald man ein Eingabefeld mit der Maus anwählt, soll dieses leicht eingefärbt werden. Du kannst die Farbe selber wählen. Verwende dazu einen EventListener für den `focus`-Event statt `click`. ## Websitefarbe Öffne beim Laden der Seite ein Prompt-Fenster, das den Besucher nach einer Farbe (auf englisch) fragt. Setze dann die Hintergrundfarbe auf die eingegebene Farbe. Wenn du Code direkt beim Laden ausführen willst, kannst du den Code einfach direkt ins script-Tag schreiben. Eine Eingabe von `blue` soll z.B. die Farbe auf blau setzen. ## Anfang der Seite Baue auf deiner Website für ein beliebeiges Element am unteren Rand der Seite ein, dass ein Klick darauf zum Anfang der Seite scrollt. Du kannst mit einem EventListener einen eigenen Befehl definieren, in dem du statt ein Element zu verändern den Code `window.scrollTo(0, 0)` verwenden.
