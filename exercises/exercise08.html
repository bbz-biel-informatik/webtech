# Scrollen Baue auf deiner Website ein paar Elemente ein, die bei einem Klick drauf das Browserfenster an einen anderen Ort auf der Website bewegen. Du kannst beispielsweise unten auf der Seite einen Link einbauen, der an den Beginn der Website springt. # Sound Baue auf deiner Website einen kleinen Musikplayer ein. Er soll mindestens einen Play/Pause-Knopf haben, der die Musik kontrollieren kann. Wenn der Player funktioniert, kannst du noch eine Lautstärkeregelung einbauen. Verwende dazu `audio.volume`. Du kannst die Lautstärke in einer Zahl zwischen 0 und 1 angeben, z.B. `audio.volume = 0.5`. # Animiertes Scrollen Animationen sind viele kleine Bewegungsschritte, die schnell nacheinander ausgeführt werden. Der Mensch kann die einzelnen Schritte nicht mehr auseinanderhalten und sieht sie als Bewegung. Mit dem Befehl `setTimeout` kannst du einen Befehl in einer gewissen Zeit in der Zukunft ausführen. Das kannst du verwenden, um eine Scroll-Animation zu bauen: ```js function scrollen() { window.scrollBy(0, 3) // 3 Pixel nach unten scrollen setTimeout(scrollen, 16) // in 16ms (60x pro Sekunde) weiterscrollen } ``` Dieses Codestück führt den Befehl `scrollen` aus, der a) 3px nach unten scrollt b) sich selber nach 16ms wieder ausführt Für das menschliche Auge sieht das dann aus wie eine Animation. Baue den Scroll-Code auf deiner Website so um, dass das Scrollen animiert ist. # Fussnoten Baue in einen Text deiner Website eine Referenz auf eine Fussnote ein, ähnlich wie auf Wikipedia: <h4>Haus<sup>1</sup></h4> Ein Klick auf die Zahl soll zu der entsprechenden Fussnote scrollen. Die Fussnote soll einen Link beinhalten, der wieder nach oben scrollt. # Lösungen ## Scrollen ```js var topButton = document.querySelector("#top") topButton.addEventListener('click', rauf) function rauf() { window.scrollTo(0, 0) } ``` ## Sound ```js var audio = new Sound("http://mysounds.com/sample.mp3") var playButton = document.querySelector("#play") var pauseButton = document.querySelector("#pause") playButton.addEventListener('click', play) function play() { audio.play() } function pause() { audio.pause() } ``` ## Animiertes Scrollen ```js var scrollButton = document.querySelector("#scroll") scrollButton.addEventListener('click', scrollen) function scrollen() { window.scrollBy(0, 3) if(window.scrollY < 200) { // nur weiterscrollen, wenn noch nicht 200px gescrollt setTimeout(scrollen, 16) } } ```
