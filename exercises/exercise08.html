# Scrollen Schreibe eine Website, die drei grosse Überschriften hat und dazwischen ein wenig Text. Füge dann am oberen Rand der Seite 3 Schaltflächen hinzu, die bei einem Klick bewirken, dass der Browser zur entsprechenden Überschrift springt. <a href="sample08.html">Vorlage herunterladen</a> Um die Position eines bestehenden Elements herauszufinden, kannst du das Element auswählen: `var x = document.querySelector('#p1')` und dann die Y-Position herausfinden: `var y = x.offsetTop`. # Sound Baue auf deiner Website einen kleinen Musikplayer ein. Er soll mindestens einen Play/Pause-Knopf haben, der die Musik kontrollieren kann. Wenn du willst, kannst du noch eine Lautstärkeregelung einbauen.
