# Scrollen Schreibe eine Website, die drei grosse Überschriften hat und dazwischen ein wenig Text. Füge dann am oberen Rand der Seite 3 Schaltflächen hinzu, die bei einem Klick bewirken, dass der Browser zur entsprechenden Überschrift springt. <a href="sample08.html">Vorlage herunterladen</a> Um die Position eines bestehenden Elements herauszufinden, kannst du das Element auswählen: `var x = document.querySelector('#p1')` und dann die Y-Position herausfinden: `x.offsetTop`. # Sound Baue am unteren Rand deiner Website einen Musikplayer mit 5 Liedern ein. Der Player soll SChaltflächen zum Abspielen, Pausieren und Überspringen des aktuellen Lieds haben. Der einfachste Weg ist, ein Array mit den Liedern zu speichern, in einer separaten Variable die aktuelle Position in dieser Liste zu speichern und so den Player zu steuern. # Position Baue auf einer Website einen Knopf ein, der die Position des Besuchers abruft. Die Website soll dann in einem Nachrichtenfenster die ungefähre Distanz nach Bern anzeigen. Du kannst [diese Formel](http://stackoverflow.com/a/27943/1641070) verwenden. Die Koordinaten von Bern sind etwa 46.9479222, 7.4446085.
