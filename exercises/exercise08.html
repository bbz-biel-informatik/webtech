# Scrollen Schreibe eine Website, die drei grosse Überschriften hat und dazwischen ein wenig Text. Füge dann am oberen Rand der Seite 3 Schaltflächen hinzu, die bei einem Klick bewirken, dass der Browser zur entsprechenden Überschrift springt. <a href="sample08.html">Vorlage herunterladen</a> Um die Position eines bestehenden Elements herauszufinden, kannst du das Element auswählen: `var x = document.querySelector('#p1')` und dann die Y-Position herausfinden: `var y = x.offsetTop`. # Namen speichern Baue auf deiner Website eine Funktion ein, die den Besucher nach seinem Namen fragt und diesen im lokalen Speicher speichert. Bei den folgenden Besuchen soll die Website dann nicht mehr nach dem Namen fragen, sondern den Text "Hallo" sowie den Namen in einer Textbox anzeigen. Um zu testen, ob der Name schon gespeichert ist, kannst du den folgenden Code verwenden: ```js var n = localStorage.getItem('name'); if(n) { // name ist gespeichert, anzeigen... } else { // name nicht gespeichert, abfragen... } ``` Bonus: zeige den Text oben auf der Seite in einem Titel an (`innerHTML`, siehe letzte Woche). # Scrollposition speichern Baue in deine Website eine Funktion ein, die die aktuelle Scroll-Position speichert. Wenn der Besucher die Website verlässt und später wieder besucht, soll die Website automatisch zur vorherigen Position springen. Achtung: die Website muss relativ lang sein, damit man sieht, ob es funktioniert. Um jeweils die aktuelle Scrollposition zu erhalten, kannst du den folgenden Code verwenden: ```js window.addEventListener('scroll', function(e) { var y = window.scrollY; // weiterer Code }); ``` Du kannst den Code von Aufgabe 2 verwenden zum Testen, ob schon ein Scrollwert gespeichert ist.
