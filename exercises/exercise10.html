# Aufgaben Local Storage ## Namen Frage den Besucher deiner Website beim ersten Besuch nach seinem Namen und speichere ihn. Zeige bei jedem weiteren Besuch den Namen in einem Fenster an. Frage den Besucher nur, wenn du nicht schon einen Namen gespeichert hast. ## Scrollposition Speichere die Scrollposition des Benutzers und springe an diese Stelle, wenn der Besucher deine Website später wieder öffnet. Um herauszufinden, wann und wohin der Benutzer gescrollt hat, kannst du den folgenden Code verwenden: ```js document.addEventListener('scroll', onScroll) function onScroll() { var yPosition = window.scrollY ... } ``` ## Musikplayer Baue den Musikplayer, bei dem du die Adresse der Musikdatei selber eingeben kannst, so um, dass die Adresse gespeicher wird und die Musik beim nächsten Besuch automatisch abgespielt wird. ## Farbe Speichere die Farbe, die der Benutzer als Hintergrundfarbe eingegeben hat, ebenfalls und färbe die Website damit automatisch ein.
