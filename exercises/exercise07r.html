# Wiederholung und Sounds ## Klickzähler Baue auf deiner Website einen Klickzähler ein. Sobald ein Besucher entweder ein Bild, eine Überschrift oder einen Textabsatz anklickt, soll der Zähler um 1 hochzählen und den aktuellen Stand anzeigen (entweder mit `alert` oder in einem `<input type="text">`). Zur Erinnerung: `<input type="text" value='0'/>` ```js var textFeld = document.querySelector('input'); textFeld.value = 12; ``` ## Diashow Baue eine Diashow in deine Website ein. Du darfst den Beispielcode verwenden. Zusatzaufgaben: * Schaffst du es, weitere Knöpfe zu programmieren, die direkt an den Anfang/ans Ende der Bildserie springen? * Kannst du neben den Bildern eine Anzeige programmieren, die anzeigt, welches Bild gerade gezeigt wird (z.B. "3 von 5")? Tipp: du kannst den Inhaltstext eines Elements mit `.innerHTML` setzen, zum Beispiel mit `meinElement.innerHTML = x + ' von 5'`. ## Musikplayer Programmiere auf deiner Website einen Musikplayer und füge mindestens eine Abspielen/Pause-Taste ein. Zusatzaufgabe: * Erstelle eine Playlist, die mehrere Lieder enthält. Füge dann einen "Weiter" bzw. "Zurück"-Knopf ein, die in der Playlist hin- und herspringen. Du kannst mehr oder weniger den gleichen Code wie für die Diashow verwenden. Pass auf, dass die Knöpfe andere IDs oder andere Klassen haben als in der Diashow.
