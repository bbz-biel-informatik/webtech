# Attribute lesen ## Klickzähler Baue auf deiner Website Code ein, der bei jedem Klick auf ein Bild einen Zähler hochzählt und den aktuellen Stand in einem `alert()`-Fenster anzeigt. ## Linkadresse Baue Code ein, der bei jedem Link auf der Website beim Klick darauf die Adresse des Links in einem `alert`-Fenster anzeigt. ## Bildadresse Baue Code ein, der bei jedem Bild auf der Website beim Klick darauf die Adresse des Bildes in einem `alert`-Fenster anzeigt. ## Linkabfrage Baue auf deine Website Code ein, der bei einem Klick auf einen beliebigen Link fragt, ob man den Link wirklich öffnen will. Die Frage kannst du so programmieren: ```js function frage(e) { var antwort = confirm('Wirklich öffnen?') if(antwort == false) { e.preventDefault() } } ``` ## Bildpopup Schreibe Code, der bei einem Klick auf ein Bild dieses in einem neuen Fenster öffnet. Dazu musst du im Klick-Handler das `src`-Attribut des Bildes lesen und dann diese Adresse öffnen. Ein neues Fenster kannst du mit dem folgenden Code öffnen: ```js window.open('http://google.com') ``` ## Bildpopup 2 Schreibe den Popup-Code um so dass das Fenster nicht in einem neuen Tab geöfnet wird, sondern im aktuellen Fenster gross angezeigt wird. Ein erneuter Klick soll es wieder verkleinern. Tipp: Setze beim Klicken die `position`, `left`, `top` und `width/height`-Eigenschaften des Bildes, und setze sie beim zweiten Klick wieder zurück. # Lösungen ## Klickzähler ```js var counter = 0 var bilder = document.querySelectorAll('img') for(var i = 0; i < bilder.length; i = i + 1) { bilder[i].addEventListener('click', zaehlen) } function zaehlen() { counter = counter + 1 alert(counter) } ``` ## Linkadresse ```js var links = document.querySelectorAll('a') for(var i = 0; i < links.length; i = i + 1) { links[i].addEventListener('click', adresse) } function adresse() { alert(this.getAttribute('href')) } ``` ## Bildadresse ```js var bilder = document.querySelectorAll('img') for(var i = 0; i < bilder.length; i = i + 1) { bilder[i].addEventListener('click', adresse) } function adresse() { alert(this.getAttribute('src')) } ``` ## Linkadresse ```js var links = document.querySelectorAll('a') for(var i = 0; i < links.length; i = i + 1) { links[i].addEventListener('click', frage) } function frage(e) { var antwort = confirm('Wirklich öffnen?') if(antwort == false) { e.preventDefault() } } ``` ## Bildpopup ```js var bilder = document.querySelectorAll('img') for(var i = 0; i < bilder.length; i = i + 1) { bilder[i].addEventListener('click', popup) } function popup() { window.open(this.getAttribute('src')) } ```
