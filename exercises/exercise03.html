# Übungen Schleifen ## Zusammenfassung ```js for(var i = 1; i < 10; i = i + 1) { alert(i); } ``` ## Aufgaben ### Dreierreihe * Schreibe ein Programm, das die Zahlen der Dreierreihe anzeigt. ### FizzBuzz * Schreibe ein Programm, das durch die ersten 20 ganzen Zahlen durchläuft und jeweils * "Fizz" anzeigt, wenn die Zahl durch 3 teilbar ist * "Buzz" anzeigt, wenn die Zahl durch 5 teilbar ist * "FizzBuzz" anzeigt, wenn die Zahl durch 3 und 5 teilbar ist * Die Zahl selber anzeigt, wenn sie weder durch 3 noch durch 5 teilbar ist ### Summe * Schreibe ein Programm, das die Summe der ersten 20 Zahlen berechnet (1 + 2 + ... + 20). Zur Kontrolle: das Resultat sollte 210 sein. ### Rückwärts * Schreibe ein Programm, das von 10 aus rückwärts bis Null zählt und jeweils die Zahlen anzeigt. ## Lösungen ### Dreierreihe ```js for(var i = 1; i <= 20; i = i + 3) { alert(i); } // ODER for(var i = 1; i <= 20; i = i + 1) { alert(i * 3); } ``` ### FizzBuzz ```js for(var i = 1; i <= 20; i = i + 1) { if(i % 3 == 0 && i % 5 == 0) { alert("FizzBuzz"); } else if(i % 3 == 0) { alert("Fizz"); } else if(i % 5 == 0) { alert("Buzz"); } else { alert(i); } } ``` ### Summe ```js var summe = 0; for(var i = 1; i <= 20; i = i + 1) { summe = summe + i; } alert(summe); ``` ### Rückwärts ```js for(var i = 10; i >= 0; i = i - 1) { alert(i); } ```
