# HTML + JS ## Aufgabe 0 Füge Javascript-Code auf deine Website ein, der bei einem Klick auf einen Knopf die Nachrichtenbox "klick" anzeigt. ## Aufgabe 1 Nimm eine bestehende Website von dir (z.B. die aus HTML/CSS) und füge Knöpfe hinzu, die die Farben der verschiedenen Elemente ändern. ## Aufgabe 2 Ändere deinen Code so ab, dass die Website immer noch einigermassen gut aussieht. Wenn du z.B. den Hintergrund dunkel machst, solltest du die Schrift dafür heller machen. Das Ziel ist, dass man die Website umschalten kann zwischen zwei Versionen, z.B. einer hellen und einer dunklen. ## Aufgabe 3 Schreibe Code, der auf deiner Website allen Bildern einen roten 1px breiten Rahmen gibt. ## Aufgabe 4 Schreibe Code, der auf deiner Website jedem zweiten Bild einen blauen und jedem anderen zweiten einen roten Rahmen macht. ## Aufgabe 5 Schreibe Code, der auf deiner Website den Textabschnitten den Text rot färbt, aber nur, wenn mehr als 3 Abschnitte existieren (zur Erinnerung: die Anzahl Elemente in einem Array kannst du mit `myArray.length` abrufen).
