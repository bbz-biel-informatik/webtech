## Kollisionen Es muss in jedem Schritt des Game-Loops überprüft werden, ob der Spieler mit irgend einem anderen Objekt kollidiert. Dazu brauchen wir eine Schleife, die durch alle Objekte durchgeht und überprüft, ob der Spieler und das Objekt sich überlappen. Um die Grösse von Spieler oder Objekt zu finden, kann der Befehl `var position = $('.player').get(0).getBoundingClientRect()` verwendet werden. So erhält man eine Variable, in der die Position aller Ränder des Objekts gespeichert sind: ```js var players = $('.player') // mehrere player-Objekte var pos = players.get(0).getBoundingClientRect() // pos => { x: 10, y: 8, width: 24, height: 24, top: 8, right: 60, bottom: 58, left: 10 } ``` ![overlapping](images/overlapping.png) Wann überlappen sich zwei Objekte?
