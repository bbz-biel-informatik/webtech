# Animiertes Scrollen Animationen sind viele kleine Bewegungsschritte, die schnell nacheinander ausgeführt werden. Der Mensch kann die einzelnen Schritte nicht mehr auseinanderhalten und sieht sie als Bewegung. Mit dem Befehl `setTimeout` kannst du einen Befehl in einer gewissen Zeit in der Zukunft ausführen. Das kannst du verwenden, um eine Scroll-Animation zu bauen: ```js function scrollen() { window.scrollBy(0, 3) // 3 Pixel nach unten scrollen setTimeout(scrollen, 16) // in 16ms (60x pro Sekunde) weiterscrollen } ``` Dieses Codestück führt den Befehl `scrollen` aus, der a) 3px nach unten scrollt b) sich selber nach 16ms wieder ausführt Für das menschliche Auge sieht das dann aus wie eine Animation. Baue den Scroll-Code auf deiner Website so um, dass das Scrollen animiert ist. # Fussnoten Baue in einen Text deiner Website eine Referenz auf eine Fussnote ein, ähnlich wie auf Wikipedia: <h4>Haus<sup>1</sup></h4> Ein Klick auf die Zahl soll zu der entsprechenden Fussnote scrollen.
