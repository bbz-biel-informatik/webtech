## Aufgaben ### Prozent Schreibe ein Programm, das den Benutzer nach 2 Zahlen fragt, und ausrechnet, wie viele Prozent der ersten Zahl die zweite ist. Es soll z.B. für die Zahlen 4 und 3 das Resultat "75%" anzeigen. ### Kreisradius Schreibe ein Programm, das den Benutzer nach einer Zahl fragt, und ausrechnet, wie gross der Flächeninhalt eines Kreises mit diesem Radius wäre. Zur Erinnerung: A = &pi; * r<sup>2</sup>. ### Zahlen raten Schreibe ein Programm, das eine Zufallszahl zwischen 0 und 100 auswählt, und dich raten lässt, was die Zahl ist. Das Programm soll anzeigen, ob du "zu tief", "zu hoch" oder "richtig" geraten hast. Du sollst höchstens 10 mal raten dürfen. Tipp: Mit `var a = Math.round(Math.random() * 100)` wird eine Zahl zwischen 0 und 100 in `a` gespeichert ### Mehrfache von 3 oder 5 Die Summe aller Mehrfachen von 3 oder 5, die kleiner als 10 sind, ist 23 (3 + 5 + 6 + 9). Finde die Summe aller Mehrfachen von 3 oder 5, die kleiner als 1000 sind. ### Fibonacci Schreibe ein Programm, dass die ersten 20 Fibonacci-Zahlen anzeigt. Zur Erinnerung: eine Fibonacci-Zahl ist die Summer der beiden vorderen Elemente der Fibonacci-Reihe (1, 1, 2, 3, 5, 8, 13, ...) ### Primzahlen Schreibe ein Programm, das automatisch die ersten 10 Primzahlen findet. ## Lösungen ### Prozent ```js var z1 = prompt("Zahl 1?"); var z2 = prompt("Zahl 2?"); alert(z2 / z1 * 100 + '%'); ``` ### Kreisradius ```js var r = prompt("Radius?"); alert(Math.PI * r * r); ``` ### Zahlen raten ```js var num = Math.round(Math.random() * 100); for(var i = 0; i < 10; i = i + 1) { var r = prompt('Zahl?'); if(r < num) { alert('zu klein'); } else if(r > num) { alert('zu gross'); } else { alert('richtig!'); break; } } ``` ### Mehrfache von 3 oder 5 ```js var summe = 0; for(var i = 0; i < 100; i = i + 1) { if(i % 3 == 0 || i % 5 == 0) { summe = summe + i; } } alert(summe); ``` ### Fibonacci ```js var vorletzte = 1; var letzte = 1; for(var i = 0; i < 18; i = i + 1) { var nächste = vorletzte + letzte; alert(nächste); vorletzte = letzte; letzte = nächste; // zahlen 1 nach hinten schieben } ``` ### Primzahlen ```js function istPrimzahl(num) { if(num < 2) { return false; } for (var i = 2; i < num; i = i + 1) { // für jede Zahl wird getestet, ob sie durch irgendeine Zahl teilbar ist if(num % i == 0) return false; } return true; } for(var i = 0; i < 100; i = i + 1){ if(istPrimzahl(i)) { alert(i); } } ```
