# Elemente generieren und löschen ## Divs Baue auf deiner Website ein Knopf ein, der bei jedem Klick ein rotes, quadratisches Div generiert und einfügt ## Bilder Ändere deinen Knopf so ab, dass er nicht ein div, sondern ein beliebiges Bild einfügt. Du kannst HTML-Attribute mit dem Befehl `setAttribute()` setzen. Um z.B. das `src`-Attribut eines Bildes zu setzen, kannst du den Code `meinBild.setAttribute("src", "http://www.googl.com/logo.jpg")` für dein generiertes Element verwenden. Das ergibt das HTML-Element `<img src="http://www.google.com/logo.jpg"/>` ## Schneesimulation Baue auf deiner Website eine Funktion ein, die Schneefall simuliert. Die Website soll dazu bei jedem Knopfdruck an einer zufälligen Position ein kleines weisses div generieren. Um Elemente zufällig auf der Seite zu positionieren kannst du den folgenden Code verwenden (wenn x dein Div ist): ``` x.style.position = 'absolute' x.style.left = Math.random() * 100 + '%' x.style.top = Math.random() * 100 + '%' ``` ## Schnee 2 Schreibe den Code zur Schneesimulation so um, dass der Schnee automatisch (ohne Klick) erscheint. Um Code zu wiederholen kannst du diesen Befehl nehmen: window.setInterval(function() { var x = document.createElement('div') .... }, 50) Der Code wird so alle 50 ms (20 Mal pro Sekunde) ausgeführt. ## Websitevorschau Versuche eine Website zu bauen, die mehrere Links zu externen Seiten und einen einzigen iframe beinhaltet. Sobald ein Besucher auf einen Link klickt, soll dieser nicht geöffnet, sondern im iframe angezeigt werden. Um eine Website in einem iframe anzuzeigen, kannst du dessen `src`-Attribut auf die Website-URL setzen. Achtung: einige Websites lassen es nicht zu, dass sie in einem iframe geladen werden (z.B. Google). In diesem Fall siehst du in der Javascript-Konsole eine entsprechende Meldung. ## Navigation 1997-Style Baue eine Website, die oben eine Navigation mit Untermenupunkten anzeigt. Das Menu soll die folgende Struktur haben: ``` Home Seiten - Über mich - Gästebuch Sitemap ``` Sobald Benutzer mit der Maus über den Eintrag "Seiten" fahren, soll ein Untermenu ausklappen, das dessen Unterseiten anzeigt. Am einfachsten geht das, wenn du im HTML-Code alle Menupunkte definierst und sie per JS anzeigst und versteckst. ## Diashow Baue eine Website, die eine Diashow beinhaltet. Die Diashow soll zum nächsten Bild wechseln, sobald die Besucher auf das Bild klicken. Am einfachsten geht das, indem du ein Array mit den Internet-Adressen der Bilder speicherst und beim Klick jeweils ein neues Bild-Element generierst, das die entsprechende Adresse als `src`-Attribut hat. Dann positionierst du das Bild gleich wie das vorherige. ## Lösungen ## Divs ```js var knopf = document.querySelector('button'); knopf.addEventListener('click', div); function div() { var d = document.createElement('div'); d.style.backgroundColor = 'red'; d.style.width = '200px'; d.style.height = '200px'; var b = document.querySelector('body'); b.appendChild(d); } ``` ## Bilder Siehe Lösung "divs" ```js d.setAttribute('src', 'http://example.com/meinbild.jpg'); ``` ## Schnee ```js var knopf = document.querySelector('button'); knopf.addEventListener('click', schnee); function schnee() { var x = document.createElement('div'); x.innerHTML = '*'; x.style.position = 'absolute'; x.style.left = Math.random() * 100 + '%'; x.style.top = Math.random() * 100 + '%'; var b = document.querySelector('body'); b.appendChild(x); } ``` ## Schnee 2 ```js // alle 50ms einmal den Befehl schnee ausführen window.setInterval(schnee, 50); function schnee() { var x = document.createElement('div'); x.innerHTML = '*'; x.style.position = 'absolute'; x.style.left = Math.random() * 100 + '%'; x.style.top = Math.random() * 100 + '%'; var b = document.querySelector('body'); b.appendChild(x); } ``` * [Aufgabe Vorschau](http://jsfiddle.net/vh1ga7q4/) * [Aufgabe Navigation](http://jsfiddle.net/9vajbj4p/) * [Aufgabe Diashow](http://jsfiddle.net/4m8x2sz8/)
